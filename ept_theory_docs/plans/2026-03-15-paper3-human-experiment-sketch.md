# Paper 3: Human Experiment Sketch (DRAFT)

**Status:** Early design — needs IRB approval before any data collection

## Why existing data failed

The CRA (Compound Remote Associates) dataset has **independent trials** — each problem is solved
separately, there's no learning trajectory across trials. EPT predicts phase-transition signatures
in systems that *gradually acquire* a skill, not in one-shot problem solving.

The THINGS-EEG dataset showed object recognition (single trial, not learning).
The velocity-RT effect did not replicate across subjects.

## What the ideal experiment looks like

**Task requirements:**
1. Subjects must LEARN something across trials (not solve independent problems)
2. The "correct structure" must be known a priori (like Fourier for grokking)
3. Trial-by-trial behavioral data must be available (RT, accuracy, strategy)
4. An independent "aha" marker (self-report button press)
5. Some subjects should show sudden learning, others gradual (natural variation)

**Best candidate: Artificial Grammar Learning (AGL)**

Subjects see letter strings generated by a finite-state grammar (e.g., MXRVXT).
After exposure, they classify NEW strings as grammatical/ungrammatical.
The "correct structure" is the grammar itself — known by construction.

**Why AGL is ideal for EPT:**
- There IS a learning trajectory (exposure phase → test phase → improvement over time)
- The task structure (grammar) is known and computable
- Some subjects report "aha" moments when they suddenly "see" the pattern
- Trial-by-trial accuracy and RT are standard measurements
- Very well-studied — lots of prior data for comparison
- Easy to implement (no equipment beyond a computer)

## Experimental design

### Phase 1: Exposure (100 trials)
- Present grammatical strings, subject classifies as grammatical/not
- No feedback initially (first 20 trials)
- Feedback starts at trial 21 (the "seed")
- Every trial: record RT, response, confidence

### Phase 2: Feedback removal (50 trials)
- Continue classifying, but remove feedback (the "hysteresis test")
- Does performance persist without the seed?

### Phase 3: Novel strings (50 trials)
- Test on strings from SAME grammar but never seen before
- Tests genuine rule learning vs memorization

### Self-report
- After every 10 trials: "rate your understanding on 1-10"
- At end: "was there a moment when you suddenly understood the pattern?"
- If yes: "approximately which trial number?"

### Conditions (between-subjects)
- **Aligned seed**: feedback on ALL features (full grammar exposure)
- **Partial seed**: feedback only on surface features (analogous to cosθ < 1)
- **No seed**: no feedback at any point (control — analogous to WD=0)

### Measurements
- Primary: trial-by-trial accuracy trajectory
- Secondary: RT trajectory
- Velocity proxy: |Δ(rolling accuracy)| or KL divergence of strategy
- F_task: accuracy on novel strings (tests true grammar learning, not memorization)
- Hysteresis: accuracy in Phase 2 vs Phase 1 (does it persist without feedback?)

### Sample size
- Power analysis for medium effect (d=0.5): N=34 per group
- 3 groups: N=102 total
- Each session ~30 minutes
- If recruiting from class: feasible in 1-2 weeks

### Analysis (pre-registered)
1. Compare accuracy trajectories: does "aligned seed" group show sharper transitions?
2. Velocity analysis: is |Δ(rolling accuracy)| higher around the self-reported aha trial?
3. Hysteresis: does Phase 2 accuracy decay less in the "aligned seed" group?
4. Dose-response: does partial seed show intermediate behavior?
5. Novel-string transfer: is F_task (novel accuracy) higher for aligned seed?

### EPT predictions (falsifiable)
1. "Aligned seed" group will show faster and sharper accuracy transitions
2. Self-reported aha moments will coincide with velocity peaks (within ±3 trials)
3. Performance will persist in Phase 2 (hysteresis) for subjects who had aha moments
4. No-feedback group will show lower F_task (less transfer to novel strings)
5. Velocity at aha trial will be higher than velocity at random matched trials (permutation test)

### What would falsify EPT
- If all three groups learn at the same rate → seed doesn't matter
- If aha moments don't coincide with velocity peaks → EPT timing prediction fails
- If performance decays equally in Phase 2 for all groups → no hysteresis
- If novel-string accuracy is equal across groups → no genuine structure learning

## Implementation

Simple web-based task (JavaScript/HTML). No special equipment.
Can run on lab computers or students' laptops (remote possible).

**Code effort**: ~1 week to implement the task
**Data collection**: ~1-2 weeks with student recruitment
**Analysis**: ~1 week

**Total**: ~1 month if IRB approval is fast (or exempt for educational research)

## IRB considerations

- Minimal risk (computer-based task, no deception beyond standard AGL protocol)
- Likely exempt if framed as educational research
- Informed consent required
- Data anonymized (subject IDs only)
